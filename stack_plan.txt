The plan of creating a program for the realisation of stack

Function names:
    1. isEmpty - checks if stack is empty
    2. isFull - checks if stack is full
    3. push - adds a single element to the top of the stack, increasing size by 1
    4. pop - removes a single element from the top of the stack, decreasing size by 1
    5. show - shows the entire contents of the stack 
    6. peek - prints the element from the top of the stack in stdout
